# Bujji 

## Project Overview
Bujji is a Snapchat-like website that allows users to share images and messages with friends. This project will be developed using a full-stack application approach where the frontend is built with React and the backend is powered by Node.js and Express.

## Features
- User Authentication
- Image and message sharing
- Real-time notifications
- Responsive design

## Technologies Used
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB (or any other)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Nannraj-2006/Bujji.git
   ```
2. Install dependencies for both frontend and backend.

## Usage
- Start the backend server
- Start the frontend application
